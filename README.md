# 📦 DAYS - Dijital Askı Yönetim Sistemi

Bu proje, **Yönetim Bilişim Sistemleri Görsel Programlama II** dersi dönem ödevi kapsamında geliştirilmiş, C# Windows Forms ve MS SQL Server tabanlı bir otomasyon yazılımıdır.

## 🚀 Öne Çıkan Özellikler
- **ADO.NET & MS SQL Server Entegrasyonu:** Tamamen dinamik veri akışı.
- **SQL Injection Koruması:** Güvenli parametreli sorgu mimarisi.
- **Rol Tabanlı Yetkilendirme (RBAC):** `Yönetici`, `Bağışçı` ve `Alıcı` rollerine göre dinamik değişen akıllı arayüz tasarımı.
- **Yönetici Kontrol Merkezi:** Sistemdeki aktif ürünlerin ve kayıtlı kullanıcıların yan yana listelendiği gelişmiş çift tablolu panel.
- **Dinamik Stok Yönetimi:** Alıcı ürünü rezerve ettiğinde anlık olarak veri tabanından düşme ve arayüzü otomatik tazeleme.
- **Güvenli Oturum Yönetimi:** Hesaplar arası geçişi sağlayan "Güvenli Çıkış" sistemi.
